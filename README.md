# Project name

Louvre API

# Description

Personal project in which testing order methods of API of the Louvre Museum web-app is carried out

# Technologies

Java 11, maven 2.22.2, JUnit 4.13.2, allure 2.23.0, rest-assured 5.3.1, gson 2.10.1

# Clean up and run tests

`mvn clean test`

# Allure report

`mvn allure:serve`
